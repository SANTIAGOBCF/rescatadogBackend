# Quick Install
## Create a virtual env:
`python3 -m venv zz_env`

Activate virtual env

## Install all dependencies:

`pip install -r requirements.txt`


## Run django server:
`python manage.py runserver`


## Visit the api:
`http://127.0.0.1:8000/api/docs`

## Visit admin:
`http://127.0.0.1:8000/admin`
